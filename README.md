### 主要機能
  - Authentication
    - メール認証
  - Payment
    - Stripe/Paypal 決済機能
  - Admin
    - 商品管理(追加、編集、削除)
    - ユーザー管理(追加、編集、削除)
    - 注文管理(発送ステータス変更)
  - 商品検索機能
  - ページネーション
  
### 技術スタック
  - React.js
  - Redux (Redux-toolkit)
  - React-bootstrap
  - Node.js/Express
  - ライブラリ
    - Stripe
    - PayPal
  - データベース & 開発環境
    - MongoDB
    - Git/CLI
    - Github
    - Postman
